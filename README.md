# CodeAlpha_StockPortfolio

## Description
This is a basic Python program that acts as a Stock Portfolio Tracker. It takes user inputs for stock names and quantities, calculates the total investment value using hardcoded prices, and allows users to save the output to a text file.

## Features
- Validates user inputs (checks for correct stock symbols and integers).
- Calculates the total investment.
- Saves the portfolio summary to a `.txt` file.

## How to Run
1. Clone or download the project files.
2. Open the `.py` or `.ipynb` file in your Python environment.
3. Run the script and type a stock name (e.g., AAPL) along with its quantity when prompted.
4. Type `done` when you finish adding stocks.
